# 📚 Campus X Practice: Data Science Fundamentals

**Campus_X_Practice** is a hands-on learning repository where I practice and explore core Data Science libraries in detail by following the **Campus X YouTube Playlist**.

---

## 🚀 What's Inside

This repo covers in-depth practice of the following libraries:

- **NumPy** — Array operations, indexing, slicing & mathematical computations
- **Pandas** — DataFrames, data cleaning, manipulation & analysis
- **Matplotlib** — Data visualization & plotting techniques
- **Seaborn** — Statistical data visualization & advanced plots

---

## 🛠️ Tech Stack

- **Language**: Python 🐍
- **Environment**: Jupyter Notebook
- **Libraries**: NumPy, Pandas, Matplotlib, Seaborn
